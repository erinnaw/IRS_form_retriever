part1.py
1) In the console terminal, run part1.py.
2) The program will prompt for input.
3) Write a list of form values, each separated by a space. Exclude the word "form". 
An example input would be, "w-2 w-2g w-3 990-w"
4) The script will return a list of dictionary in JSON format. The list will be printed on the console. 

part2.py
1) In the console terminal, run part2.py.
2) The program will prompt for input.
3) Write the input in the format of "<Form Value> <Start Year> <End Year>". Exclude the word "form". 
For example, "w-2 2000 2008".
4) The script will create a subdirectory within the project folder with the name of the form number.
All requested PDF forms will be downloaded into the subdirectory.
